# @rxweave/store-memory

In-memory EventStore adapter for RxWeave — PubSub fan-out, for tests and ephemeral local runs.

## Install

```bash
bun add @rxweave/store-memory
```

See the [RxWeave monorepo](https://github.com/Derek-X-Wang/rxweave) for the full design spec, quickstart, and package catalog.
