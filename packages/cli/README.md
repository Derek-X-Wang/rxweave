# @rxweave/cli

The `rxweave` CLI — init, dev, emit, stream, query, inspect, registry. AI-agent-friendly (NDJSON I/O, no prompts).

## Install

```bash
bun add @rxweave/cli
```

See the [RxWeave monorepo](https://github.com/Derek-X-Wang/rxweave) for the full design spec, quickstart, and package catalog.
