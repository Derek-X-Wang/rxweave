# @rxweave/reactive

Stream helpers for filtering and transforming RxWeave event streams (whereType, byActor, bySource, withinWindow).

## Install

```bash
bun add @rxweave/reactive
```

See the [RxWeave monorepo](https://github.com/Derek-X-Wang/rxweave) for the full design spec, quickstart, and package catalog.
