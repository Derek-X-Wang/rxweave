# @rxweave/store-file

JSONL file-backed EventStore adapter for RxWeave with fsync + cold-start recovery.

## Install

```bash
bun add @rxweave/store-file
```

See the [RxWeave monorepo](https://github.com/Derek-X-Wang/rxweave) for the full design spec, quickstart, and package catalog.
