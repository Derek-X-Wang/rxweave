# @rxweave/llm

LLM-backed agents for RxWeave via Vercel AI SDK — tool calling, Effect Schema args, event emission.

## Install

```bash
bun add @rxweave/llm
```

See the [RxWeave monorepo](https://github.com/Derek-X-Wang/rxweave) for the full design spec, quickstart, and package catalog.
