# @rxweave/store-cloud

Bearer-authenticated cloud EventStore client for RxWeave over @effect/rpc NDJSON.

## Install

```bash
bun add @rxweave/store-cloud
```

See the [RxWeave monorepo](https://github.com/Derek-X-Wang/rxweave) for the full design spec, quickstart, and package catalog.
