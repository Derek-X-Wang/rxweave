# @rxweave/server

Local HTTP event-stream server for RxWeave. Hosts `@rxweave/protocol`'s RpcGroup via `@effect/rpc` NDJSON. Embed in an app (same process as your UI) or run standalone via `rxweave serve`.

## Install

```bash
bun add @rxweave/server
```

See the [RxWeave monorepo](https://github.com/Derek-X-Wang/rxweave) for the full design spec.
